## 💡 關鍵發現

### 讓賽中的詭異現象:

1. **馬號1的逆轉**
   - March 統計: 1號 Hit Rate 38.6% (A) / 34.2% (B) ← 避免
   - Race 4 現實: 1號獲勝,負磅只有125(不是最高)
   - **推論**: 在讓賽中,1號可能有特殊優勢(輕負磅+策略位置?)

2. **負磅最低的馬反而失利** 
   - Race 4: 12號負磅最低(116)但排位最好(1檔)...卻跑第5
   - March 統計: 12號都被A/B選中
   - **推論**: 讓賽中"太輕的負磅"可能無法競爭 (需要一定重量才能carry策略)

3. **負磅最高的馬反而成功**
   - Race 4: 2號負磅最高(133)但仍然跑第2
   - Race 1: 1號負磅最高(134)但能跑Top 3
   - **推論**: 高評分+重負磅的馬在讓賽中反而有advantage (被看低 = 被低估)

## 🎯 修正後的融合策略

### 新規則框架:

```python
IF 賽制 == "讓賽":
    # 讓賽邏輯: 相信A/B的共識勝過個人馬號規則
    IF A和B有共識 (都選同一馬號):
        → 必選 (無須質疑)
    ELSE:
        → 用負磅作為signal: 重負磅(>130) 優先相信B

ELSE:  # 普通賽
    # 普通賽邏輯: 馬號特定規則優先
    IF 馬號 == 2: 相信A (51.4%)
    IF 馬號 == 3,4: 相信B (51.6%, 51.9%)
    IF 馬號 == 1: 避免 (38.6%)
```
