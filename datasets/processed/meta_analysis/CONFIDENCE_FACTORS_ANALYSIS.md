## 🎯 Part 2: 共識馬命中率 (最重要!)

### 核心發現

```
成功場次: 共識馬命中率平均 55.1%
  → 過半嘅共識馬都中咗頭3
  
失敗場次: 共識馬命中率平均 36.7%
  → 只有約1/3嘅共識馬中咗

差異: +18.4pp ← 最強嘅預測信號!
```

### 為乜會咁?

**成功場次嘅邏輯:**
- A同B都選到正嘅馬 (共識馬)
- 呢啲馬多數都會進到頭3 (55% 命中率)
- 所以最後3隻馬有2隻以上會中

**失敗場次嘅邏輯:**
- A同B嘅選擇可能唔夠aligned
- 共識馬中率只有36.7%
- 加上獨選馬唔夠力,結果就<2中

### 實例對比

**成功例 - 2026-03-25 HV R2 (3中!):**
```
共識馬: [3, 1, 9] (3隻馬)
  - 馬3: A:46.5% + B:51.6% = 49.0% (高)
  - 馬1: A:38.6% + B:34.2% = 36.4% (低)
  - 馬9: A:60.0% + B:50.0% = 55.0% (高)
  → 共識平均: 47.3%
  
預測: [9, 3, 12]
實際: [3, 9, 12]
結果: 3中 ✅ (因為共識馬多數都中)
```

**失敗例 - 2026-03-04 HV R9 (0中):**
```
共識馬: [1, 4, 3] (3隻馬)
  - 馬1: A:38.6% + B:34.2% = 36.4% (較低)
  - 馬4: A:50.0% + B:51.9% = 50.9% (中)
  - 馬3: A:46.5% + B:51.6% = 49.0% (中)
  → 共識平均: 45.3% (看起來唔差)
  
預測: [11, 4, 3]
實際: [7, 9, 6]
結果: 0中 ❌ (共識馬都冇中!)
```

## 🏆 Part 8: 信心場次定義 (最終方案)

### 【高信心 / HIGH CONFIDENCE】 - 預期 70%+ 中2隻

必須符合以下**全部**條件:

```
✅ 共識馬 >= 3 隻
✅ 共識馬中嘅高命中率馬 (>50%) >= 2 隻
✅ 共識馬命中率 > 55% (理想)
✅ A同B high-aligned (至少3隻共識)

特徵:
  - A同B嘅選擇高度一致
  - 共識裡面有明星馬 (9/11/2 for A, 3/4 for B)
  - 期望: 3/3 or 2/3
  
實例: 2026-03-25 HV R2
  - 共識: [3, 1, 9] (3隻)
  - 高命中馬: [3, 9] (2隻>45%)
  - 結果: 3中 ✅
```

### 【中信心 / MEDIUM CONFIDENCE】 - 預期 55-70% 中2隻

必須符合以下**全部**條件:

```
✅ 共識馬 >= 2 隻
✅ 共識馬中嘅高命中率馬 (>50%) >= 1 隻
✅ 共識馬命中率 > 50%
✅ 至少有1隻明星馬在共識

特徵:
  - A同B基本aligned (2-3隻共識)
  - 共識馬質量OK
  - A獨選backup available
  
實例: 2026-03-04 HV R1
  - 共識: [1, 9] (2隻)
  - 高命中馬: [9] (1隻60%)
  - 結果: 2中 ✅
```

### 【低信心 / LOW CONFIDENCE】 - 預期 <55% 中2隻

任何不符合上述條件嘅:

```
❌ 共識馬 < 2 隻
❌ 冇高命中率馬在共識
❌ 共識馬命中率 < 45%
❌ A同B alignment弱 (1隻or唔align)

特徵:
  - A同B分歧較大
  - 共識馬質量一般
  - 風險較高
  
實例: 2026-03-04 HV R9
  - 共識: [1, 4, 3] (3隻,但質量差)
  - 高命中馬: 0隻 (都係<50%)
  - 結果: 0中 ❌
```

## 🎯 Part 9: 實際應用 (4 Steps)

### Step 1: 計算共識馬 (A ∩ B)

```python
consensus = set(A_picks) & set(B_picks)

```

### Step 2: 查看命中率

```python
high_rate_horses = 0
for horse in consensus:
    rate_A = march_A_rates.get(horse, 0.40)
    rate_B = march_B_rates.get(horse, 0.40)
    combined_rate = rate_A * 0.65 + rate_B * 0.35  # Method Y weight
    if combined_rate > 0.50:
        high_rate_horses += 1
        print(f"  {horse}: {combined_rate:.1%} ⭐ (high)")
    else:
        print(f"  {horse}: {combined_rate:.1%}")
```

### Step 3: 判斷信心等級

```python
if len(consensus) >= 3 and high_rate_horses >= 2:
```
