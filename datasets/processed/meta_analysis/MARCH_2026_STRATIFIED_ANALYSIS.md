## 🎯 Part 4: 讓賽中的1號馬現象

### 4.1 Race 4 的啟示 (HV讓賽)

```
根據April 2日 Race 4結果:
  - 1號贏馬 (配率2.4 - 市場看好)
  - 2號第二 (配率2.7 - 市場看好)
  - 3號第三 (配率9.0)

Predictor表現:
  - A選[1,2,3,12]: 3/4 hit → 75% ✅
  - B選[1,3,2,12]: 3/4 hit → 75% ✅
  - 我的"避免1號"融合: 2/3 hit → 66.7% ❌
```

### 4.2 修正認知

**March數據顯示:**
```
1號整體Hit Rate: 
  - 從March 77場統計: 1號選44次,命中17次 = 38.6% (A視角)
  - 从March 77場統計: 1號選38次,命中13次 = 34.2% (B視角)
  
但在讓賽中可能不同:
  - Race 4 HV讓賽: 1號獲勝 ✅
  - 讓賽中"輕負磅"的1號可能有特殊advantage
```

**結論**: 
- "避免1號"規則僅對普通賽有效
- 在讓賽中,若A/B都選1號,應該相信共識
- 不應該試圖"改進"A/B的選擇

## 💡 Part 5: 修正後的融合策略

### 5.1 讓賽特定邏輯 (Handicap Race Logic)

```python
IF race_type == "讓賽":
    IF A和B都選同一馬號:
        → 必選 (共識優先)
        → 預期Hit Rate: 65-75%
    
    ELIF A選該馬 AND B未選:
        IF 馬號 in [2, 9, 11]:  # A的強馬
            → 相信A (Hit Rate: 50-60%)
        ELSE:
            → 謹慎(Hit Rate: 44-50%)
    
    ELIF B選該馬 AND A未選:
        IF 馬號 in [3, 4]:  # B的強馬
            → 相信B (Hit Rate: 50-52%)
        ELSE:
            → 謹慎(Hit Rate: 40-48%)

預期融合Hit Rate: 55-65%
```

### 5.2 普通賽特定邏輯 (Regular Race Logic)

```python
IF race_type == "普通賽":
    # 馬號特定規則
    IF 馬號 == 2:
        → 相信A (51.4%)
    
    ELIF 馬號 in [3, 4]:
        → 相信B (51.6%, 51.9%)
    
    ELIF 馬號 in [9, 11]:
        → 相信A (60%)
    
    ELIF 馬號 == 1:
        → 避免 (38.6%-34.2%)
    
    ELSE:
        → 相信A (略微優勢)

預期融合Hit Rate: 60-65%
```
